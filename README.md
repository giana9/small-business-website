<h1 align="center">Bobby's Bakery (Small Business Website)</h1>

A mobile‑friendly website that showcases different baked goods and allows users to update the main headline with their favorite treat for added interactivity. This project demonstrates basic HTML structure, CSS styling, and JavaScript DOM manipulation.

---

<h2 align="center">☆ Project Structure ☆</h2>

- **index.html** – Main webpage containing the layout and content  
- **style.css** – Styles, colors, spacing, and responsive design  
- **cc_7.js** – Updates the headline based on user input  
- **images/** – Folder containing images used in the service cards (bread, cupcakes, cookies)

---

<h2 align="center">☆ Features ☆</h2>

- **Hero Section** – A welcoming banner introducing Bobby’s Bakery with a customizable headline.
- **Services Section** – Three product cards:
  - Bread  
  - Cupcakes  
  - Cookies  
- **Each card includes:**
  - A title  
  - A description  
  - An image    
- **Interactive Headline** – Prompted with listing their favorite sweet treat, users can type into an input box and click **Update** to transform the main headline.

---

<h2 align="center">☆ JavaScript Functionality ☆</h2>

The JavaScript file:
- Selects the input box, button, and headline  
- Listens for a button click  
- Replaces the headline text with the user’s input (if not empty)

This demonstrates basic DOM manipulation and event handling.

---

<h2 align="center">☆ Responsive Design ☆</h2>

The website includes a mobile‑friendly layout:
- Service cards stack vertically on smaller screens  
- Text and spacing adjust for readability  
- Flexbox ensures clean alignment across devices

---

<h2 align="center">☆ How to Run the Project ☆</h2>

1. Download ZIP
2. Open folder
3. Extract all files
4. Open **index.html** in any browser
